# Let's play

Plataforma web para convidar amigos para uma jogatina. Com ela você pode criar salas com definição de horários e jogo, além de criar e gerenciar ligas e torneios! 🏆

## Stacks

- ReeactJS
- NodeJS
- Tailwind
- PostgreSQL
